# ⚡ SMS Crypt Pro - Szybki Start

## 🚀 W 5 minut od zera do działającej aplikacji!

### Krok 1: Otwórz w Android Studio (30 sekund)
```
1. Uruchom Android Studio
2. File > Open
3. Wybierz: D:\APLIKACJA oryntium.app
4. Kliknij OK
```

### Krok 2: Poczekaj na Gradle Sync (2 minuty)
Android Studio automatycznie pobierze wszystkie zależności. Poczekaj aż proces się zakończy.

### Krok 3: Uruchom aplikację (1 minuta)
```
1. Kliknij zieloną strzałkę ▶️ (lub Shift+F10)
2. Wybierz urządzenie/emulator
3. Gotowe! Aplikacja uruchomi się automatycznie
```

### Krok 4: Pierwsze użycie (1 minuta)
```
1. Utwórz PIN: 123456
2. Potwierdź PIN: 123456
3. Przyznaj uprawnienia SMS
4. Gotowe do użycia!
```

## 🔧 Szybkie budowanie APK

### Debug APK (do testowania):
```bash
.\gradlew assembleDebug
```
**Lokalizacja:** `app\build\outputs\apk\debug\app-debug.apk`

### Release APK (do publikacji):
```bash
.\gradlew assembleRelease
```
**Lokalizacja:** `app\build\outputs\apk\release\app-release.apk`

## ❓ Najczęstsze problemy

**Problem:** Gradle sync fails
**Rozwiązanie:** File > Invalidate Caches / Restart

**Problem:** Emulator nie startuje
**Rozwiązanie:** Tools > Device Manager > Usuń i utwórz nowy

**Problem:** Brak uprawnień SMS
**Rozwiązanie:** Settings telefonu > Apps > SMS Crypt Pro > Permissions
