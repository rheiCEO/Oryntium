# 🚀 Instrukcja instalacji motywu ORYNTIUM na WordPress

## 📦 Plik gotowy do instalacji

**Plik:** `oryntium-wordpress-theme-fixed.zip` (57KB) ⭐ **NOWY - POPRAWIONY!**
**Zawartość:** Kompletny motyw WordPress z obsługą 8 języków + ikony SVG + JavaScript

## 🎯 Szybka instalacja (5 minut)

### Krok 1: Przygotuj WordPress
1. Zaloguj się do panelu administracyjnego WordPress
2. Przejdź do **Wygląd → Motywy**

### Krok 2: Zainstaluj motyw
1. Kliknij **"Dodaj nowy"**
2. Kliknij **"Wgraj motyw"**
3. Wybierz plik `oryntium-wordpress-theme-fixed.zip`
4. Kliknij **"Zainstaluj teraz"**
5. Kliknij **"Aktywuj"**

### Krok 3: Skonfiguruj menu
1. Przejdź do **Wygląd → Menu**
2. Stwórz nowe menu lub użyj istniejącego
3. Przypisz do lokalizacji "Primary Menu"
4. Zapisz menu

### Krok 4: Zainstaluj wielojęzyczność (OPCJONALNE)

#### Opcja A: Polylang (ZALECANE - darmowe)
1. Przejdź do **Pluginy → Dodaj nowy**
2. Wyszukaj "Polylang"
3. Zainstaluj i aktywuj plugin
4. Przejdź do **Języki → Dodaj język**
5. Dodaj języki: Polski, Angielski, Hiszpański, Niemiecki, Hindi, Arabski, Francuski, Chiński

#### Opcja B: WPML (płatne)
1. Kup licencję WPML
2. Zainstaluj plugin WPML
3. Skonfiguruj języki

### Krok 5: Dostosuj treść (OPCJONALNE)
1. Przejdź do **Wygląd → Dostosuj**
2. Wybierz **"Opcje ORYNTIUM"**
3. Zmień tytuł, podtytuł i opis hero section
4. Kliknij **"Opublikuj"**

## 🐛 Troubleshooting:

### Motyw nie aktywuje się?
- Sprawdź czy WordPress jest w wersji 5.0+
- Sprawdź czy wszystkie pliki są w folderze `wp-content/themes/oryntium/`

### Kolory nie działają?
- Sprawdź czy `style.css` jest poprawnie załadowany
- Wyczyść cache przeglądarki (Ctrl+F5)

### Języki nie działają?
- Zainstaluj plugin Polylang lub WPML
- Sprawdź czy pliki językowe są w folderze `languages/`

### Animacje nie działają?
- Sprawdź czy JavaScript jest włączony w przeglądarce
- Sprawdź czy nie ma konfliktów z innymi pluginami
