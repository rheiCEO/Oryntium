## ⚡ Najszybszy start (2 minuty)

### Krok 1: Test lokalny
```bash
# Otwórz terminal w tym folderze i wpisz:
python -m http.server 8000
```

Otwórz przeglądarkę: **http://localhost:8000**

### Krok 2: Deploy online (Netlify - NAJŁATWIEJSZE)
1. Przejdź na **[netlify.com](https://www.netlify.com/)**
2. Zaloguj się (darmowe konto)
3. Przeciągnij cały folder **WEBSITE/** na stronę Netlify
4. **GOTOWE!** Masz live stronę!

### Krok 3: Dodaj domenę
- W Netlify: Domain settings → Add custom domain → `oryntium.app`
- Ustaw DNS jak poda Netlify
- HTTPS automatycznie!

## 📞 Potrzebujesz pomocy?

### Nie działa CSS/JS?
- Sprawdź konsolę (F12) → Console
- Sprawdź czy wszystkie pliki są w folderze

### Fonty się nie ładują?
- Sprawdź połączenie z internetem
- Google Fonts może być blokowane

### Strona nie responsywna?
- Test w DevTools (F12 → Ctrl+Shift+M)
- Sprawdź różne rozdzielczości

### 404 na live stronie?
- Sprawdź czy wszystkie pliki są wgrane
- Sprawdź wielkość liter (case-sensitive!)
