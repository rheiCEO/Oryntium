## 🚀 Jak wdrożyć?

### Opcja 1: Netlify Drop (NAJPROŚCIEJ - 2 minuty)
```
1. Otwórz netlify.com
2. Przeciągnij folder z plikami
3. Dodaj domenę: oryntium.app
4. GOTOWE!
```

### Opcja 2: GitHub Pages (5 minut)
```bash
git init
git add .
git commit -m "ORYNTIUM website"
git remote add origin https://github.com/USER/oryntium-website.git
git push -u origin main

# Włącz GitHub Pages w Settings
# Dodaj plik CNAME z treścią: oryntium.app
```

### Opcja 3: Vercel (3 minuty)
```bash
npm install -g vercel
vercel
vercel --prod
vercel domains add oryntium.app
```

📖 **Szczegóły**: Zobacz `DEPLOYMENT_GUIDE.md`
