# 🌐 ORYNTIUM.APP - Strona WWW

## 📍 Lokalizacja

Wszystkie pliki strony www znajdują się w folderze: **`WEBSITE/`**

```
WEBSITE/
├── index.html              ← Główna strona
├── styles.css              ← Style CSS
├── script.js               ← JavaScript
├── favicon.svg             ← Ikona
├── manifest.json           ← PWA manifest
├── robots.txt              ← SEO
├── sitemap.xml             ← Mapa strony
├── 404.html                ← Strona błędu
├── .htaccess               ← Config Apache
│
├── README.md               ← Główny README
├── QUICK_START.md          ← Start w 5 minut! 🚀
├── DEPLOYMENT_GUIDE.md     ← Jak wdrożyć
├── GRAPHICS_GUIDE.md       ← Jak stworzyć grafiki
├── WEBSITE_README.md       ← Pełna dokumentacja
├── WEBSITE_COMPLETE.md     ← Podsumowanie
└── package.json            ← NPM scripts
```

## 🚀 Szybki Start

### 1. Przejdź do folderu
```bash
cd WEBSITE
```

### 2. Test lokalny
```bash
# Python (już zainstalowany)
python -m http.server 8000

# Otwórz: http://localhost:8000
```

### 3. Deploy online
Przeczytaj: **`WEBSITE/QUICK_START.md`** (5 minut do wdrożenia!)
