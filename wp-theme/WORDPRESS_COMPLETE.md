## 🚀 JAK ZAINSTALOWAĆ? (3 metody)

### ⭐ Metoda 1: Przez Panel WordPress (NAJŁATWIEJSZE - 2 minuty)

```
1. Zaloguj się do WordPress: twoja-domena.pl/wp-admin
2. Wygląd → Motywy → Dodaj nowy → Wyślij motyw na serwer
3. Wybierz plik: oryntium-wordpress-theme.zip
4. Kliknij: Zainstaluj teraz → Aktywuj
5. GOTOWE! 🎉
```

### 📂 Metoda 2: Przez File Manager Hostinger (3 minuty)

```
1. hpanel.hostinger.com → Files → File Manager
2. Przejdź do: public_html/wp-content/themes/
3. Upload folder oryntium/ (lub rozpakuj ZIP)
4. WordPress → Wygląd → Motywy → Aktywuj ORYNTIUM
5. GOTOWE! 🎉
```

### 🔧 Metoda 3: Przez FTP (dla zaawansowanych)

```
1. Połącz się przez FileZilla (FTP)
2. Wgraj folder oryntium/ do /wp-content/themes/
3. WordPress → Wygląd → Motywy → Aktywuj
4. GOTOWE! 🎉
```

📖 **Szczegółowa instrukcja**: `oryntium/HOSTINGER_INSTALL.md`

## 🐛 Troubleshooting

### CSS/JS się nie ładują?
1. Wyczyść cache przeglądarki (Ctrl+Shift+Del)
2. Ustawienia → Permalinki → Zapisz zmiany

### Motyw się nie pojawia?
1. Sprawdź czy folder to `oryntium` (nie `oryntium-master`)
2. Sprawdź ścieżkę: `/wp-content/themes/oryntium/`

### Brak tłumaczeń?
1. Zainstaluj **Loco Translate**
2. Wtyczki → Loco Translate → Motywy → ORYNTIUM

### Strona wygląda źle?
1. Sprawdź PHP version (powinno być 7.4+)
2. Wyłącz inne wtyczki (test konfliktów)

📖 **Więcej**: Zobacz `HOSTINGER_INSTALL.md` → Troubleshooting
